============
Installation
============

The recommended way to install SatIST is using `pip <https://pip.pypa.io/en/stable/>`_

.. code-block:: console

    pip install satist

The conda-forge install is built from the pypi index and takes a couple of days to
be available after a new popclass version is released.

To install from source

.. code-block:: console

    git clone https://github.com/LLNL/satist.git
    cd satist
    pip install .
